# GG Tiles | Premium Architectural Surfaces Platform

A professional, business-grade e-commerce and inventory management platform built for architectural surfaces, tiles, and marbles. This application bridges the gap between raw materials and architectural vision by combining a direct-to-consumer storefront with a proprietary 3D visualization engine.

## 🌟 Key Features

### Customer Experience
*   **Dynamic Showroom:** A responsive, real-time catalog powered by a live search engine and price-bracket filters.
*   **3D Blueprint Visualizer:** An interactive constructor allowing users to map specific tile textures directly onto digital walls, floors, and roofs.
*   **Scaling Calculator:** An integrated coverage tool that instantly calculates net square footage, 10% wastage, and total box requirements.
*   **Secure E-Commerce Flow:** User authentication, detailed material specification pages, and a dynamic checkout system with live freight and GST tax calculation.
*   **Universal Architecture:** Seamless navigation across the Showroom, About Us, Privacy Policy, and Support portals.

### Admin Central Command
*   **Secure Dashboard:** Protected staff portal for managing warehouse operations.
*   **Inventory Matrix:** Live overview of stock levels, active catalogs, and dynamic pricing.
*   **PC File Uploads:** Integrated secure file handling for uploading high-resolution material images directly from local hardware to the server.
*   **Live Database Sync:** Instant updates to pricing, stock levels, and imagery reflected globally across the platform.

## 🛠️ Tech Stack

*   **Backend:** Python, Flask, Werkzeug
*   **Database:** SQLite (Dynamic Relational Database)
*   **Frontend Design:** HTML5, Tailwind CSS, FontAwesome Icons
*   **Interactivity:** Vanilla JavaScript
*   **File Handling:** Local Server File System (`static/uploads`)
